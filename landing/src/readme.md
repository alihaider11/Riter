## Landing Page for Riter

### Develop
You need to have `node` installed. in the src folder run `npm i` and it should generate the `node_modules` folder.

use this command to build the css:
`npx @tailwindcss/cli -i .\style.css -o .\output.css`

### Deploy
Make sure to change the site url in the sitemap.xml and robots.txt after deploying the site.

the dist folder is just the needed files for deployment.
